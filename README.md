# Ember & Vine

A modern restaurant website built with React, Vite, and TypeScript.

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Motion (animations)
- Lucide React (icons)

## Features
- Categorized interactive menu (Starters, Mains, Desserts, Cocktails)
- AI-style rule-based dining assistant chatbot
- Reservation booking form
- Smooth animations and transitions
- Fully responsive design

## Getting Started
```
npm install
npm run dev
```
